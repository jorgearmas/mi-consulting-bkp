# Setup & Customization Guide

## Essential Changes

### 1. Add Your Images
- Save your facility photo as: images/hero-facility.jpg
- Update the image path in HTML if using a different filename

### 2. Update Email Address
- Search for: sam@consultwithmi.com
- Replace with: your@email.com

## Form Submission

Currently logs to browser console. To send leads by email:

### Option 1: Zapier (Free)
1. Create account at zapier.com
2. Set up webhook
3. Connect to Gmail, Outlook, or CRM
4. Get leads by email automatically

### Option 2: FormSubmit.co (Free)
1. Go to formsubmit.co
2. Add your email to HTML form action
3. Leads sent to your email

### Option 3: Your Backend
- Use PHP, Node, Python
- Store in database
- Full control

## Customization

### Change Questions
Find the form-step divs and edit question text. If you change the YES/NO pattern, update the validation logic.

### Modify Pillar Cards
Edit the text in the pillar-card divs. All cards now have the same green color (no more gold accents).

### Update Tier Pricing
Find tier-price divs and change $1,500, $2,500, $5,000.

### Add Spanish Version
Create parallel sections with Spanish content. Bilingual implementation guide available.
