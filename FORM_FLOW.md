# Qualification Form Flow

## The 4-Question Stepper

**Question 1: Manufacturer?**
- YES → Continue to Q2
- NO → Disqualified

**Question 2: SQF Focus?**
- YES → Continue to Q3
- NO → Disqualified

**Question 3: Cannabis/THC?**
- YES → Disqualified (cannot serve)
- NO → Continue to Q4

**Question 4: Budget $1.5k+?**
- YES → QUALIFIED! Show contact form
- NO → Disqualified

## Contact Form (if qualified)

Fields captured:
- Name (required)
- Email (required, validated)
- Phone (required)
- Company (required)
- Facility Location (required)
- SQF Status (optional)

## Visual Features

✓ Stepper showing progress (1→2→3→4→✓)
✓ Auto-advances between questions
✓ "Back" button to review answers
✓ "Skip for Now" to jump ahead
✓ Form validation
✓ Success confirmation
